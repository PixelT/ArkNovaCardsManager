Overwiev:
This script snippet will add additional button to the boardgamearena.com game table which will link to arknova.cards page with predefined filters based on base projects on current game / table. It will help you to quickly check cards for your actual game without need to manually select base projects.

To run this script you need to have installed browser extension which allows you to run UserScripts - Tampermonkey

It will also mark with green icon the cards on your hands that fulfil the basic projects:

Installation:
Open github gist link: https://gist.github.com/PixelT/1254f00c8ad86b25744cead5f252f09a and copy arknova.cards.bga.shortcut.js content as new Tampermonkey script / or just click the Raw button to install it automatically.
